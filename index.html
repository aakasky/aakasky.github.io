BudTrip

BudTrip is an Android application that utilizes user-provided details such as hotel specifications, number of travelers, and type of transportation to estimate the trip's budget. The application also provides suggestions for places to visit or nearby attractions. The primary objective is to optimize the trip experience by simplifying the planning process and providing accurate estimates.

LDS

LDS (Login Detail Security) is an Android application that employs Caesar Cipher to encrypt user login credentials and stores them in Firebase. The application provides a secure and private means of storing important information such as website logins, social media credentials, as well as bank and ATM card PINs and passwords in an encrypted format. This feature enables the user to conveniently and confidently manage multiple passwords without the hassle of memorizing each one.

Buddy Chat

Buddy Chat application enhances the chatting experience by providing a modern social media platform that enables users to connect with their friends and family. The application offers a real-time chat system as well as the ability to post images and videos, allowing users to stay connected and share their experiences in an engaging and interactive manner.

JITT

JITT (Just In Time Trainer) is an application designed to simplify the studying process for students by providing them with access to all necessary materials, including books and notes. In addition, the application offers several additional features, such as links to video lectures, a matrix calculator, an attendance system, quizzes, counseling sessions, career guidance orientations, and much more. These features are intended to help students excel in their studies and career.

Customized PlayFair Cipher

The application offers an enhanced security method to the Playfair Cipher, utilizing two tables for encryption - a parent table and a randomized table. For decryption, the user is required to provide a key and a relative position mapping. This mapping is with respect to the parent matrix, resulting in a more secure encryption method than the Playfair Cipher.

Filter Image

This application offers a wide range of filters that can be applied to images. Users have the option to upload an image from their files or gallery, or directly capture an image and apply filters to it. The available filters include Grey, Negative, Green, Sepia, Grey Filter Outside, Sepia Filter Outside, Grey Diagonal, Sepia Diagonal, and Sketch Filter, among others.

Online Art Gallery

The Online Art Gallery is a web application that displays various artworks using MongoDB as its back-end, and is hosted on Heroku. This platform allows users to showcase their art and also provides a space to view masterpieces from all around the world.
